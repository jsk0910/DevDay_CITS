# ➕ **앱 사용 가이드**
C-ITS는 아래와 같은 기능을 제공합니다.
1. 환자의 상태를 입력했을 때, 해당 환자에게 필요한 전문의 분야 도출
2. 해당 전문 의료 인력을 보유한 병원을 거리순으로 3개 목록화
3. 목록의 병원까지의 최적 경로를 실시간으로 도출
